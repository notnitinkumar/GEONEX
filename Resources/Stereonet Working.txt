Working
    Manual Entry

        - Uses renderplot function to render plot from the data which further uses two function drawstereonet2d function to draw stereonet and Plotplane2d/plotline2d to plot data

        - Plotplane2d function : It uses generateGreatCircle function which returns an array of {trend,plunge} and this array is further utilised by stereographicProjection function which convers these trend plunge points to x,y coordinates
